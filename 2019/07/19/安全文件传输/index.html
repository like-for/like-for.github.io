scp / sftp - 安全拷贝 / 安全文件传输

netstat - 查看网络状态
-n - numeric - 不对IP地址进行解析

-t - tcp - 查看TCP套接字
-p - process / program - 显示进程编号和程序名
-l - listen - 查看处于监听状态的套接字

TCP/IP模型 - 四层模型

传输层 - 端到端可靠传输 - TCP / UDP
网络层 - 寻址和路由 - IP/ICMP
物理链路层 - 电气特性、分帧、校验
